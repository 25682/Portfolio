Minesweeper Game

About the Project

This Minesweeper game is a fun and challenging project that aims to replicate the classic logic-based game. Players uncover cells on a grid, with the goal of avoiding mines and flagging all the mines correctly to win.

Features

  • Different difficulty levels: Easy, Medium, Hard
  • Timer to track how quickly you can solve the puzzle
  • Ability to flag and unflag potential mines
  • Randomly generated grid for each game session

GitHub Link

You can view the source code for this project on GitHub.

Technologies Used

The game is developed using JavaScript for the game logic, HTML for the structure, and CSS for styling. It demonstrates the use of event handling, DOM manipulation, and basic game mechanics in JavaScript.
