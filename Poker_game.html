Poker Game

About the Project

This Poker Game project is a digital recreation of the popular card game, allowing players to test their skills in a simulated environment. The game includes features such as betting, bluffing, and winning strategies.

Features

  • Multiplayer support with AI opponents
  • Realistic card shuffling and dealing
  • Customizable game settings and betting limits
  • Detailed statistics and game history tracking

GitHub Link

You can view the source code for this project on GitHub.

Technologies Used

The game is developed using JavaScript for the game logic and interactivity, with HTML and CSS for the layout and design. It showcases the use of advanced JavaScript techniques for game development.
